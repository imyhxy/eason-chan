### Description

创作小天王薛之谦 即将于秋末发行自己的第一张个人新歌加精选专辑，首播个人创作主打歌《未完成的歌 》CD版。《未完成的歌 》是薛之谦逾一年的创作，甚至创作早于上张专辑的同名主打歌《深深爱过你》，而为何迟迟没有发表，薛之谦 表示：“一直没有很好的词来配合这首曲，所以我一直在等，终于等到让我觉得感动的歌词。
