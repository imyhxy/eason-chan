### Description

薛之谦 2008全新个人创作专辑《深深爱过你》11月26日震撼发行

CD集合了全媒体PK热曲之专辑同名主打《深深爱过你》前世、今生双版本，各电台榜首之中国风单曲《传说》，薛之谦首度尝试韩国劲爆热舞曲目《Let You Go》《星河之役》 重新演绎上张专辑主打歌曲《苏黎世的从前》回响奥运曲目《梦开始的原点》等。DVD更收录3支情歌MV，《深深爱过你》前世、今生MV感人情节延续上演。
