13. **[Kriaaaaa \[56\]](https://music.163.com/#/user/home?id=440721983):** 果然和林夕所预料的那样，后来的陈奕迅虽然磕磕绊绊，但终究是势不可挡，一飞冲天。 在2008年，《我的快乐时代》这首歌推出了十年之后，陈奕迅在另一首歌《然后怎样》里面，终于给出了对于林夕的回应。 我的快乐时代唱烂，才领悟代价多高昂。
