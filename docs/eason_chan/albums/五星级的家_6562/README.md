### Description

有一日，苦公（陈奕迅饰）突然来港，令相处融洽的苦家掀起不少隐藏的问题；苦公与苦妈（吴君如饰）多年来的心结始终未解，苦妈与苦荣母子关系随即出现裂痕，而苦佬（谷德昭饰）与小苦妹左右为难。如何能够用爱，齐家，治国，平天下呢？一个电台广播剧，为你娓娓道来这《五星级的家》的故事，以及9首热播歌曲。
