### Description

陈奕迅终于推出广东大碟，并联同 Moving On Stage 演唱会的大乐队组成 H3M，Eason 表示与乐队成员巡回演出超过一年，十分希望能够与他们出一张唱片，他说:“希望可以与 Moving On Stage band 做一张大碟，他们每一个人都会为我写歌，是十分值得纪念，好兴奋的事情。
