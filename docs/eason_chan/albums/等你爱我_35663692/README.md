### Description

歌曲《等你爱我》，原唱陈明，是1999年电视剧《将爱情进行到底》的插曲、片尾曲之一。2010年，中国香港男歌手陈奕迅重新演绎这首歌，作为2011年电影版《将爱情进行到底》的片尾曲。而原先的陈明版《等你爱我》，也被选为这部电影的片头曲。
