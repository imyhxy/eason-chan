5. **[lyq1989 \[2013\]](https://music.163.com/#/user/home?id=109356540):** 大家听国粤双语的歌曲时往往会觉得好像粤语版比较好听，其实是因为粤语有九个声调，国语只有四个，而音谱为七个。仔细听会发现粤语歌用的音和平常说话的调基本相同，而国语却大多都要变调才可以。所以粤语歌音域更广、表现更饱满、感染力更强，陈奕迅的歌很容易就能感觉到这些特点。
