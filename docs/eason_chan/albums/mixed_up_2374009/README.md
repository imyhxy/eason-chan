## Appendix

### Description

陈奕迅现在已经贵为英皇一哥，旧东家华星唱片在此时也推出Eason首张全Remix 大碟《Mixed Up》，收录了十首Eason大热歌曲，包括‘幸福摩天轮’、‘美丽有罪’、‘专家话’、‘黑夜不再来’等的Remix 版本。不要以为《Mixed Up》是一张普通的精选辑，它是由香港多位当红电子音乐乐手包括：Minimal、梁基爵、四方果等人共同混音，呈现出完全不同的新感觉
