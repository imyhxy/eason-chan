## Appendix

### Description

继上次那张收录了新歌“一切还好”的新曲+精选《Great 5000 Secs》后，Eason 昔日所属的英皇娱乐，近日又在其转会新艺宝首张专辑《U87》推出之当，为他发行了全新一张新曲加精选专辑《Great 5000 Secs Vol.2》，继续收录三首新歌及十五首他在英皇时期的精选旧歌。
