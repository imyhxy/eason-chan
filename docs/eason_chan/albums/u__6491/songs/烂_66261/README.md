## Comments
0. **[jason3029 \[1533\]](https://music.163.com/#/user/home?id=2633021):** “烂”这首歌在我看来绝对是被严重低估的一首歌，它的魅力应不亚于被更多人狂热追捧的《浮夸》。当你听到“万物溶烂，一念无限；旷世巨钻，不过是炭”，那一刻你心底的刮躁一定不亚于听到《浮夸》里那一句“以眼泪淋花吧”。伟文写出了一个小人物的竭斯底里，而林夕则写出了睥睨万物的鞭辟入里。
