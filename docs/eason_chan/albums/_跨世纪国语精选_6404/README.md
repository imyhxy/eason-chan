## Appendix

### Description

这张《The First Eleven Years 然后呢》双CD乃首张集合Eason十年以来最完整国语精选专辑，收录了1997年至2007年间30首严精国语经典歌，包括∶“酝酿”、“预感”、“拔河”、“你的背包”、“不如这样”、“K歌之王”、“十年”、“爱情转移”等，来为Eason这十年的国语乐坛作见证。
