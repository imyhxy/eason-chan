0. **[LionHeart-Leon \[2882\]](https://music.163.com/#/user/home?id=42327424):** 这张碟主打是孤独患者，但是这张碟也仅仅红了孤独患者。。。看穿一些歌其实都挺不错的。吟游诗人其实很适合作为第二主打，但是估计是因为歌词达不到孤独患者的深度，曲调虽然带着一点爵士，但过于偏向大众口水。其实，这歌听起来还真是不错。
