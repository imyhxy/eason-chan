### Description

这张NOTHING REALLY MATTERS 是陈奕迅离开华星的最后一张大碟，喜欢陈奕迅的乐迷，应该大部份都见证着他从新秀得奖、推出第一张大碟一直到现在，短短的五年里转变之大像是意料不到，但能够拥有自己喜欢的歌手从成长到离巢，这么完整的音乐见证，你我都不应错过。
